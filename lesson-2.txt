1.
- Khởi tạo: git init (chỉ cần lần đầu khi mới tạo repo)
- Cấu hình:
    - git add <file name> : đưa 1 file vào vùng staging
    - git add . : đưa tất cả các file vào vùng staging
- Gửi commit code với mess: git commit -m "message"
